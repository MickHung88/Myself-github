WATERPROOF BUTTON AND ELECTRONIC DEVICE

Announcement No. M667600

Patent link

A waterproof button includes a circuit board assembly, a button member, an elastic sheet, a shell member, a first paste sheet and a second paste sheet. The button member is covered with the circuit board assembly. The elastic sheet is adapted to drive the button member to move. The shell member surrounds the button member. The first paste sheet surrounds an edge of the shell member. The second paste sheet is disposed between the elastic sheet and the circuit board assembly and surrounds an edge of the elastic sheet. The waterproof button avoided the water entering the electronic device form the gap between the components of the waterproof button or the gap between the waterproof button and the casing of the electronic device. An electronic device using the waterproof button is also provided.

  • Client: IP66 level button
  • Category: Dell

Water-proof door

Announcement No. TBD.

• Adopted waterproof rubber with both lateral and vertical compression interference, achieving excellent waterproof performance with IP67 or higher protection level.

• The waterproof door seal ring is assembled using snap-fit hooks. Since the seal ring is not fixed rigidly, it provides self-alignment capability, ensuring consistent sealing interference around the entire circumference without being affected by door dimensional tolerances.

• The door latch utilizes a spring arm integrated in the metal frame to provide a distinct click feeling and to secure the latch in both open and closed positions.

• A torsion spring is installed on the door rotation axis, allowing the door to automatically pop open when the user releases it.

• All components are assembled using snap-fit structures, requiring no tools for assembly.

  • Client: Waterproof door
  • Category: Dell
